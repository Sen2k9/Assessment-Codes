# How to download the source codes

1. Download this project zip file from https://github.com/Sen2k9/Assessment-Codes.git and unzip it
   or,
   use git clone https://github.com/Sen2k9/Assessment-Codes.git
  
# How to run this program

1. Go to your command prompt 
2. Change directory by using the command in your prompt
cd {path_to_folder}

3. Run the command in the shell
python3 string_manipulation.py

# How to run the test code

1. 1. Go to your command prompt 
2. Change directory by using the command in your prompt
cd {path_to_folder}

3. Run the command in the shell
python3 test_string_manipulation.py -b
